# LapSpec Analyzer

**LapSpec Analyzer** は、ノートPCの内蔵マイクを使用して、スピーカーや音響空間の周波数特性をブラウザ上で手軽に測定・解析するためのツールです。

## 概要 (Overview)

このシステムは、外部の測定用マイクやオーディオインターフェースを使わず、**「手持ちのノートPC一台で、その場の音響状態をざっくり把握する」**ことを目的として設計されています。

一般的な観賞用ビジュアライザーとは異なり、音響エンジニアリングの視点から、人間の聴覚特性に近い**対数スケール（Log Scale）**での表示と、精密なゲインコントロールを実装しています。

## 主な機能 (Features)

- **デュアル表示**: リアルタイムの波形（オシロスコープ）と周波数分布（スペクトルアナライザー）を同時に確認可能。
- **対数周波数軸**: 低域から高域まで、音楽的なバランスを確認しやすい対数グラフを採用。
- **独立したゲイン制御**: 入力段のゲイン調整に加え、表示上の倍率を個別に調整可能。
- **一時停止機能**: 特定の瞬間の波形を静止させて詳細に分析。
- **インストール不要**: ブラウザ（Chrome, Edge, Firefox等）があればどこでも動作。

## 使い方 (How to Use)

1. `oscilloB5.html`（またはリポジトリのURL）をブラウザで開きます。
2. 「マイク開始」ボタンをクリックし、マイクの使用を許可します。
3. 測定したいスピーカーから音を出し、グラフを確認します。
4. 必要に応じて「表示ゲイン」スライダーを動かし、波形が見やすい大きさに調整してください。

## 正確に測定するためのヒント (Measurement Tips)

ノートPCの内蔵マイクでより正確な特性を調べるために、以下の設定を推奨します。

- **OSの音響効果をオフにする**:
  Windowsの場合、「サウンドの設定」→「入力デバイスのプロパティ」から、**「オーディオの強化」や「ノイズ抑制」をオフ**にしてください。これらが有効だと、マイクが勝手に周波数特性を変えてしまいます。
- **マイクの向き**:
  ノートPCのマイクの位置（ベゼルの上やキーボード付近）を確認し、そこをスピーカーの正面に向けるように設置してください。

## 開発者向け (For Developers)

- **技術スタック**: HTML5 / JavaScript (Web Audio API) / Canvas API
- **カスタマイズ**: `analyser.fftSize` の値を変更することで、解像度とレスポンスのバランスを調整できます（デフォルト: 2048）。
